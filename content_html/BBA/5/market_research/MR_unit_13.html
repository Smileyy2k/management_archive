UNIT 13: TESTS OF SIGNIFICANCE

1. INTRODUCTION

This unit focuses on Inferential Statistics—the techniques used to draw conclusions about a large population based on data collected from a small sample. This is done primarily through Hypothesis Testing.

A hypothesis is an unproven statement about a phenomenon the researcher is interested in. Hypothesis Testing is a statistical procedure used to determine whether there is enough evidence in a sample to support a belief about the entire population. We start with an assumption and use statistics to see if the data proves that assumption is true, or if any change was just due to random chance.

Key Concepts

  • Null Hypothesis (H0): A statement of no difference or no effect. The researcher always attempts to reject it. (e.g., H0: There is no difference in sales between cities A and B.)
  • Alternative Hypothesis (H1): A statement that reflects the researcher's belief that some difference or effect exists. (e.g., H1: Sales in city A are higher than in city B.)
  • Level of Significance (α): The probability of Type I error (rejecting the null hypothesis when it is actually true). Typically set at 0.05 (5%).
  • P-Value: The probability of obtaining a test result at least as extreme as the one observed, assuming H0 is true. If P-Value < α, we reject H0.
  • Type I Error (α): Rejecting H0 when it is actually true (a false positive). Example: Concluding a new ad campaign worked, when in reality it didn't.
  • Type II Error (β): Failing to reject H0 when it is actually false (a false negative). Example: Concluding a new ad campaign failed, when in reality it worked.

2. METHODS OF HYPOTHESIS TESTING

The general procedure involves five main steps:

  1. Formulate H0 and H1: State the null and alternative hypotheses clearly.
  2. Select Appropriate Test: Choose the correct statistical test (e.g., t-test, Z-test, Chi-Square).
  3. Choose Significance Level (α): Determine the threshold for rejecting H0 (usually 0.05).
  4. Calculate Test Statistic: Compute the statistical value (e.g., t or Z score) using the sample data.
  5. Make Decision: Compare the calculated P-Value to α to decide whether to reject or fail to reject H0.

3. PARAMETRIC TESTS

A parametric test is a statistical test that makes specific assumptions about the population data, such as assuming the data comes from a normal distribution. These tests require continuous data and are considered more statistically powerful.

A. Z-Test (One Sample)

  • Objective: To test if the population mean (µ) is significantly different from a hypothesized value (µ0).
  • Use Case: When the sample size is large (n ≥ 30) or the population standard deviation (σ) is known.

B. t-Test (One Sample)

  • Objective: Same as Z-test, but used when the population standard deviation (σ) is unknown and the sample size is small (n < 30).

C. Independent Samples t-Test (Two Samples)

  • Objective: To determine if the means of two unrelated groups are significantly different.
  • Example: Is the average satisfaction score of Male customers significantly different from that of Female customers?

D. Paired Samples t-Test (Two Samples)

  • Objective: To determine if the means of two related groups are significantly different. This is used for "before and after" measurements on the same sample.
  • Example: Did the average willingness-to-buy score significantly increase after the same group of consumers viewed a new product demonstration?

4. CHI-SQUARE (X²) ANALYSIS

Chi-Square analysis is a Non-Parametric Test used for nominal or ordinal data. A non-parametric test does not assume data follows a specific probability distribution, making it "distribution-free."

The Chi-Square (X²) test is used to test the Goodness of Fit (one-way analysis) and the Association or independence (two-way analysis) between two categorical variables. It determines if the observed pattern of frequencies is significantly different from the expected pattern.

A. Chi-Square Goodness of Fit (One-Way)

  • Objective: To determine if the observed frequency distribution for a single categorical variable matches a theoretically expected distribution.
  • Example: We expect 25% of customers to prefer each of four brands (A, B, C, D). Does our actual survey data fit this equal expectation?

B. Chi-Square Test of Association (Two-Way)

  • Objective: To test whether there is a statistically significant association between two categorical variables in a cross-tabulation table.
  • Null Hypothesis (H0): The two variables are independent (no association).
  • Example: Is there an association between Region (North, South, East, West) and Product Purchased (Product X, Product Y)?
